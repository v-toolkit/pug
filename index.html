Pagero Universal Generator

Build Pagero Universal Format invoice & credit-note XML

Set the country, template and default direction once in Setup, then add scenarios. Each scenario is one document, and every party shows the identifier fields its country actually uses — SIREN and SIRET for France, GSTIN and PAN for India, VAT and registration numbers elsewhere.

Setup

Applies to every scenario. Choose the country, set the default direction, and provide the country example as the template.

Pick an example below to load it automatically, or copy one from GitHub and paste it in: select a country to get its example link
